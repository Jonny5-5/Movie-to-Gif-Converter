# Movie-to-Gif-Converter
Transform movies into small Gifs  


### Setup
You will need to install the moviepy package as shown below.
```
pip install moviepy
```


### File Structure
The outputs will be placed in the `output` directory
```
Movie-to-Gif-Converter
├───output
│   ├───gifs
│   └───gifs_subs
│
└───movie_gif.py
```
